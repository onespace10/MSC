# Usage
```
docker build -t daya123/nginx5 .

build -t daya123/nginx6 .

$ vi Dockerfile
$ docker build -t daya123/nginx6 . 
$ docker images
$ docker push daya123/nginx6
$ cd MSA
$ git add -A
$ git commit
$ git push
$ touch README.md
$ vi README.md
$ git add -A
$ git commit -m "README.md added"
$ git push
$ pwd
/c/Users/user/MSC
$ ls -al
total 50
```
```
$git add –A <--- All
$git commit –m “add a” <--- add a
$git push <-- Ubuntu upload, git web download
$git pull <-- git web upload, Ubuntu download
```
